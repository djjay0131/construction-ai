# Conference Submission Planning

This folder holds target-conference research for the construction.ai project.

## Purpose

We are casting a wide net across top-tier venues that could plausibly fit one or more of these angles:

- construction informatics / construction automation
- engineering verification, validation, and uncertainty quantification (VVUQ)
- software engineering / systems / digital twins
- AI, knowledge graphs, agentic systems, or applied ML for engineering workflows
- high-quality industry or translational construction venues when clearly top tier

## Folder Structure

- `matrix.md` — master comparison table across all candidate conferences
- `<conference-slug>.md` — one markdown file per conference

## Scoring

Each conference should be given a **Hotness** rating from 1 to 5:

- **5** = excellent fit, strong submission target
- **4** = very good fit, worth serious consideration
- **3** = plausible but may require angle adjustment
- **2** = stretch venue or weak fit
- **1** = probably not worth pursuing

## Notes

For each venue we want to capture:

- official conference name
- sponsoring organization / community
- why it counts as top tier
- relevant tracks or paper types
- abstract / full paper / proposal / workshop deadlines
- conference dates and location
- whether the current deadline has passed
- whether another paper type may still be open
- likely next-cycle timing if the current cycle is closed
- recommended framing for our work

## Status

This folder currently contains scaffolding only. Live deadline and CFP research should be verified from official conference sites before making submission decisions.
